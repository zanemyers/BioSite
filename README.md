# BioSite

A personal portfolio and informational website built with **React**, **Tailwind CSS**, and **Radix UI Themes**.  
The site showcases my projects, resume, skills, updates, and provides informational pages like Terms of Service and Privacy Policy.

## Features

- Responsive and modern design  
- Dark and light mode support using Radix UI Themes  
- Dynamic routing with **React Router v6**  
- Project and skill showcase sections  
- Updates section for personal and professional posts  
- Terms of Service and Privacy Policy pages  
- Custom 404 page that respects the selected theme  

---

## Tech Stack

- **Frontend:** React, TypeScript, Tailwind CSS  
- **UI Components:** Radix UI Themes  
- **Routing:** React Router v6  
- **Hosting:** Render (static site deployment)  

---

## Pages

- **Home** – Introduction and overview  
- **Resume** – Skills, experience, and education  
- **Projects** – Portfolio projects with descriptions  
- **Updates** – Personal and professional updates  
- **Terms of Service** – Terms of using this site  
- **Privacy Policy** – Information about privacy practices  
- **404 Not Found** – Custom error page for unknown routes
